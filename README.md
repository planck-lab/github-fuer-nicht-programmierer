# GitHub für Nicht-Programmierer

Eine kostenlose Mini-Lernreise für alle, die nicht programmieren — aber besser zusammenarbeiten wollen.

## Was ist das?

GitHub ist kein Programmier-Tool. Es ist ein Kollaborations- und Versionierungstool. Diese Lernreise zeigt dir, wie du GitHub nutzen kannst — ohne eine einzige Zeile Code.

## Für wen?

- Projektmanager die Versionschaos kennen
- Gründer die mit Dev-Teams arbeiten
- Wissenschaftler die an Dokumenten kollaborieren
- Alle die vor der "Programmier-Wand" stehen
