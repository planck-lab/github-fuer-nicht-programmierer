Wusstest du, was alles auf GitHub liegt?

Ich dachte jahrelang: GitHub = Programmierung. Nichts für mich.

Dann habe ich das hier entdeckt:

📜 **Deutsche Bundesgesetze** — Alle Gesetze und Verordnungen, versioniert und durchsuchbar. Jede Änderung an jedem Paragraphen nachvollziehbar.

🔬 **COVID-19-Forschungsdaten** — Die Johns Hopkins University hat DIE zentrale Pandemie-Datenquelle auf GitHub gepflegt. Über 30.000 Sterne.

🏢 **Ein Firmenhandbuch für 2.000 Mitarbeiter** — GitLab hat Onboarding, Gehälter, Prozesse komplett öffentlich und versioniert.

🍳 **Kochrezepte** — Ja, wirklich. Mit Änderungshistorie, falls jemand die Salzmenge anpasst.

Keines davon ist Code. Alles ist Zusammenarbeit an Dokumenten.

GitHub ist kein Programmier-Tool. Es ist das Werkzeug, das dein Datei-Chaos löst.
